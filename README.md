O projeto foi realizado no capítulo IV do Ignite, no curso disponibilizado pela Rocketseat.
No projeto foram utilizados algumas bibliotecas, como:
- [x] Chakra-ui
- [x] React Hook Form
- [x] ApexCharts
- [x] NextJs
- [x] Yup
